# Company RAG Chatbot

A simple Retrieval-Augmented Generation (RAG) chatbot built using FastAPI, ChromaDB, Sentence Transformers, and Groq.

## Features

- Scrapes website content
- Converts content into embeddings
- Stores embeddings in ChromaDB
- Retrieves relevant chunks using semantic search
- Generates grounded answers using Llama 3

## Tech Stack

- Python
- FastAPI
- ChromaDB
- Sentence Transformers
- Groq API
- Playwright
